## listAll

### Example Usage

```typescript

const supabase = new Supabase({
  bearer: "<YOUR_BEARER_TOKEN_HERE>",
});

async function run() {
  const result = await supabase.snippets.listAll({});

  // Handle the result
  console.log(result);
}

run();
```
